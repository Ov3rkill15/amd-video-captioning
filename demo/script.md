# Demo Script — AMD ACT II Video Captioning

<!-- Isi H-1 sebelum deadline (10 Juli), jangan dadakan di hari terakhir. -->

## Alur demo (draft)
1. Tunjukkan video input singkat (contoh di `demo/samples/`)
2. Jalankan `python main.py --input <video> --output demo/samples/output.json`
3. Tunjukkan frame yang diekstrak + caption per frame di terminal
4. Buka `output.json` hasil akhir, highlight kualitas narasi
5. (Opsional) Bandingkan dengan run di AMD Developer Cloud / MI300X kalau sempat
